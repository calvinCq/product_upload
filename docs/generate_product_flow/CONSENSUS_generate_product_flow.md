# CONSENSUS: 产品描述生成与添加商品流程

## 需求描述
基于sample_product_description.txt文件中的培训课程信息，生成产品描述和图片，并将产品添加到微信小店。

## 技术实现方案
1. 修改config_manager.py，确保正确加载钱多多API配置
2. 修改product_generator.py，支持从文件读取产品描述并生成产品数据
3. 利用现有的钱多多API图片生成功能生成课程相关图片
4. 修改product_uploader.py，实现产品图片上传和商品添加功能
5. 修改main.py，整合整个流程

## 技术约束
1. 仅使用Python语言
2. 仅修改src目录下的核心代码
3. 使用钱多多API进行图片生成
4. 使用微信小店API进行商品添加
5. API密钥等敏感信息使用.env文件管理

## 验收标准
1. 能成功从sample_product_description.txt读取产品描述
2. 能成功调用钱多多API生成产品图片
3. 能成功将产品信息和图片添加到微信小店
4. 整个流程无错误，生成的产品信息完整
5. 代码修改符合项目现有规范

## 任务边界
1. 不创建新的测试文件
2. 不修改非核心代码文件
3. 不使用火山相关代码
