   - 待办：用户需要根据实际使用的火山大模型，在.env文件中设置正确的模型名称

2. **API密钥配置**
   - 状态：已在.env.example中添加相关配置
   - 待办：用户需要在.env文件中设置有效的火山大模型API密钥
   - 操作指引：复制.env.example为.env文件，将VOLCANO_API_KEY=your_api_key_here替换为实际的API密钥

3. **微信小店API配置**
   - 状态：已在.env.example中添加相关配置
   - 待办：用户需要在.env文件中设置有效的微信小店API凭证
   - 操作指引：复制.env.example为.env文件，将WECHAT_APPID和WECHAT_APPSECRET替换为实际的凭证

## 功能限制说明

1. **支持的功能**
   - 已实现：生成商品图片功能
   - 已实现：上传商品到微信小店功能
   - 已移除/未实现：其他非必要功能

2. **模型选择说明**
   - 如果未配置model_name，系统将默认使用"imagegen"模型
   - 请确保配置的模型名称与火山大模型API支持的模型列表一致
   - 不同模型可能支持不同的参数和功能，请根据模型文档进行相应调整
